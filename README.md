# Denie : auto-reply-chatbot

_**Denie is an automatic interactive chat bot help you learn something new**_
_**and give you entertainment by letting you play and tell you funny joke**_

# How to use Denie:

- Run Denie chat boat in terminal
- Provide it your name
- Have fun...
- Yes Ask whatever you want from the available options

# How it works

- Give it a prompt
- It will process your prompt
- Make a meaningingful prompt
- Perform task if found prompt meaningful
- Ask user if they are happy with response
- If yes, save the prompt(query), user name, time, and response in DB file
- If no generate another response
- Repeat same process

# What Denie can do?

- It can tell you a joke
- It can help you improve your vocabulary by teaching you new word
- It can entertain you by playing some game
- It can help you perform calculation (provide calculator)
- It can tell you today's day, date, time, etc
- It can help you generate a strong password
